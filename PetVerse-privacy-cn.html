PetVerse 隐私政策

最后更新：2026-07-08 · 开发者：PetVerse Team

1. 摘要

PetVerse 是一款完全本地运行的虚拟养宠游戏：

  • 我们不运营任何存储用户数据的后端服务器。
  • 我们不要求账号、登录、邮箱或手机号。
  • 我们不收集、不上传、不出售、不共享、不追踪任何个人信息。
  • 我们不集成任何第三方分析、广告、崩溃上报或营销 SDK。
  • 我们不使用 Cookie、IDFA 或任何广告标识符。

2. 存储在设备上的数据

以下数据仅通过 Apple 官方 SwiftData 本地数据库和 UserDefaults 保存在您的设备上：

  • 宠物信息：名字、物种、外形、等级、心情、饥饿、清洁、精力、羁绊值、成就、心愿、探险进度、金币余额、相册、家园装饰等。
  • 使用偏好：音效、触感反馈、语言、新手引导状态。
  • 您从相册选择或用相机拍摄的照片（仅用于设置您本人或宠物的头像）：仅在本机存储，不会上传。

如需清除，卸载 App 即可，iOS 会一并删除本地数据。

3. 可选 AI 图像生成（默认关闭）

PetVerse 提供一项可选功能：用阿里云 DashScope 生成风格化的 3D 形象或杂交后代形象。

  • 该功能默认关闭，仅当您在"设置 → AI 图像"中粘贴自己的 API Key 后才启用。
  • 点击"生成"时，仅将文本 prompt（杂交场景还包含您已选中的亲本图片）通过 HTTPS 发送到 dashscope.aliyuncs.com，使用您自己的 API Key 调用。生成的图片会保存到本地。
  • PetVerse 本身不接收、不存储、不观察这些请求。
  • 您随时可在设置中删除 API Key 停止使用该功能。

4. 相机与相册权限

App 只在您点击对应功能时才请求：

  • 相册访问——选取图片作为个人或宠物头像。
  • 相机访问——拍照作为个人或宠物头像。

即使拒绝，App 其它部分照常使用。

5. 儿童隐私

PetVerse 不会主动收集任何人（含 13 周岁以下未成年人）的个人信息。所有数据均不离开设备，适合家庭、儿童放心使用。

6. 变更

如未来有修订，我们会更新页顶"最后更新"日期，并在 App Store Connect 关联的相同链接下发布新版本。

7. 联系我们

如有疑问：support@petverse.app
