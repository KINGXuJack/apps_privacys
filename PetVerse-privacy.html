PetVerse Privacy Policy

Last updated: 2026-07-08 · Developer: PetVerse Team

1. Summary

PetVerse is an offline virtual pet game. It runs entirely on your device.

  • We do not operate any backend server that stores your data.
  • We do not require an account, sign-in, email, or phone number.
  • We do not collect, transmit, sell, share, or track any personal information.
  • We do not embed third-party analytics, advertising, crash reporting, or marketing SDKs.
  • We do not use cookies, IDFA, or any advertising identifiers.

2. Data stored on your device

The following data is created and stored only on your device (via Apple's SwiftData local database and UserDefaults):

  • Your pets: names, species, appearance, level, mood, hunger, cleanliness, energy, bond, achievements, wish list, adventure progress, coin wallet balance, album entries, and cottage decorations.
  • User preferences: sound, haptics, language, onboarding state.
  • Photos you pick from the Photo Library or capture with the camera only for setting your profile / pet avatar. These images are stored locally and never uploaded.

You can erase all of the above at any time by deleting the app; iOS removes all local data.

3. Optional AI image generation (opt-in)

PetVerse contains an optional feature that can generate a stylized 3D pet portrait or a hybrid-baby image using Alibaba Cloud DashScope.

  • This feature is off by default and only activates if you paste your own API key in Settings → AI Image.
  • When you tap "Generate", the text prompt (and, for hybrid generation, the reference pet photos you already selected) is sent over HTTPS to dashscope.aliyuncs.com using your own API key. The response image is saved locally.
  • PetVerse itself does not receive, store, or observe the traffic.
  • You can stop using this feature at any time by removing the API key from settings.

4. Camera and Photo Library permissions

The app requests two iOS permissions only when you tap a feature that needs them:

  • Photo Library — to let you pick an image for your profile or pet avatar.
  • Camera — to let you take a new photo for your profile or pet avatar.

Denying either permission simply disables that button; the rest of the app works normally.

5. Children's privacy

PetVerse does not knowingly collect personal information from anyone, including children under 13. As no data leaves the device, PetVerse is safe for family / child use.

6. Changes

If we ever change this policy we will update the "Last updated" date above and publish the new version at the same URL that App Store Connect links to.

7. Contact

Questions: support@petverse.app
